### 使用方法：

请打开QuanX 配置文件-编辑，找到［policy］字段，并在相应策略组中末尾，<br>加上img-url=https://raw.githubusercontent.com/sngxpro/icons/main/genshin/图标名称.png<br> 注意同样要用英文逗号与前面句子隔开，并在逗号后面空一格

其中的“图标名称”可对应下面图标下方的标注查看，也可自行查看上方的文件。
